Windows Azure Tools para Microsoft Visual Studio

Para habilitar la compatibilidad con la compilación de aplicaciones web y servicios Web escalables en Windows Azure, instale Windows Azure Tools para Microsoft Visual Studio.

Windows Azure Tools para Microsoft Visual Studio amplía Visual Studio para habilitar la creación, compilación, depuración, ejecución y empaquetado de aplicaciones web y servicios Web escalables en Windows Azure. La instalación incluye el SDK de Windows Azure.

Visite la página de descarga para consultar los requisitos del sistema, los sistemas operativos compatibles y las instrucciones de instalación.

Nota: el proyecto que contiene esta página solo proporciona instrucciones para instalar el conjunto de herramientas requerido.
